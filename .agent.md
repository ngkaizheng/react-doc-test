# Agent UI workflow enforcement

When modifying or adding UI in this repository, the agent MUST:

1. Query the MCP component catalog using `mcp_react-ui-cata_get_component_spec` for any named component intended for use.
2. If the component exists, implement using the returned XML contract and props exactly.
3. If the component does not exist, report back to the user and ask whether to create a new composite using available MCP components.
4. Do not proceed with custom markup for UI components before completing steps 1–3.

This file is used to prevent hallucination and ensure adherence to the UI-expert "No-Guess" workflow.
